API Documentation

1. GET /

Check if the application is running successfully.

Request: Method: GET, URL: /

Response: Status Code: 201, Body: { "message": "App running successfully" }

2. POST /uploadmenu

Upload a new menu item.

Request: Method: POST, URL: /uploadmenu

Response: Status Code: 201, Body: { "message": "Menu uploaded successfully" }

3. GET /getallmenu

Retrieve all menu items.

Request: Method: GET, URL: /getallmenu

Response: Status Code: 200, Body: Array of menu items.

4. GET /getmenu

Retrieve menu items by category and subcategory.

Request: Method: GET, URL: /getmenu?category={category}&subcategory={subcategory}

Response: Status Code: 200, Body: Array of menu items matching the provided category and subcategory.

5. PUT /updatemenu/:id

Update a menu item by ID.

Request: Method: PUT, URL: /updatemenu/:id

Response: Status Code: 200, Body: { "message": "Item updated successfully", "updated_data": { updated menu item object } }

6. DELETE /deletemenu/:id

Delete a menu item by ID.

Request: Method: DELETE, URL: /deletemenu/:id

Response: Status Code: 200, Body: "Item deleted successfully"

7. POST /addcategory

Add a new category.

Request: Method: POST, URL: /addcategory

Response: Status Code: 201, Body: { "message": "Category added successfully" }

8. GET /getcategory

Retrieve all categories.

Request: Method: GET, URL: /getcategory

Response: Status Code: 200, Body: Array of categories.

9. PUT /updatecategory/:id

Update a category

Request: Method: PUT, URL: /updatecategory/:id

Response: Status Code: 200, Body: { "message": "Category updated successfully", "updated_data": { updated category object } }

10. DELETE /deletecategory/:id

Delete a category by ID.

Request: Method: DELETE, URL: /deletecategory/:id

Response: Status Code: 200, Body: "Category deleted successfully"

11. POST /addsubcategory

Add a new subcategory.

Request: Method: POST, URL: /addsubcategory

Response: Status Code: 201, Body: { "message": "Subcategory added successfully" }

12. GET /getsubcategory/:category

Retrieve all subcategories of a specific category.

Request: Method: GET, URL: /getsubcategory/:category

Response: Status Code: 200, Body: Array of subcategories.

13. GET /getsubcategorybyname/:subcategory_name

Retrieve subcategories by name.

Request: Method: GET, URL: /getsubcategorybyname/:subcategory_name

Response: Status Code: 200, Body: Array of subcategories matching the provided name.

14. PUT /updatesubcategory/:id

Update a subcategory by ID.

Request: Method: PUT, URL: /updatesubcategory/:id

Response: Status Code: 200, Body: { "message": "Subcategory updated successfully", "updated_data": { updated subcategory object } }

15. DELETE /deletesubcategory/:id

Delete a subcategory by ID.

Request: Method: DELETE, URL: /deletesubcategory/:id

Response: Status Code: 200, Body: "Subcategory deleted successfully"
